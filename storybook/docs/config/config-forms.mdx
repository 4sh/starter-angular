# Configuration : catégorie Forms

Constantes de la famille **forms** (`$form-*`). Chaque variable hérite par défaut d'une valeur
globale et peut être surchargée pour **tous** les contrôles de formulaire à la fois.

> Fichier source : `src/styles/src/settings/_ui-config.scss` · exposé via `@use 'utils'`.
> Consommée par : `ui-checkbox`, `ui-radio`, `ui-toggle`, `ui-field` (donc `ui-input`,
> `ui-input-mask`, `ui-input-number`). Retour au
> [fonctionnement & sommaire](?path=/docs/components-configuration--docs).
