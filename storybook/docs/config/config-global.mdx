## Motion (`$motion-*`)

Timing partagé du **système de motion** (presets d'entrée/sortie + transitions d'état).
Consommé par `utils.motion-transition()`, `base/_motion.scss` et la directive `[uiMotion]`.
Surchargeable par élément via les custom properties `--ui-motion-*`. Voir **Foundations → Motion**.
