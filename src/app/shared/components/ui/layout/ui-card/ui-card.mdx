# ui-card

Conteneur **flexible** qui compose des régions optionnelles : média, en-tête
(titre + sous-titre), contenu et pied. Chaque région n'est rendue que si elle
porte du contenu — aucun wrapper vide, donc aucun espacement fantôme.

Surface neutre (tokens `global`) : fond `--global-default-surface-default`, texte
`--global-high-content-default`, bordure `--global-default-stroke-default`, ombre
`--shadow-default-md` (variante `elevated`), rayon `--radius-lg`. Light **et** dark
via les modes de variables.

## Système de slots

Le cas courant ne demande **aucun import** : le contenu par défaut passe par
`<ng-content>`, et le titre / sous-titre acceptent les inputs `header` / `subheader`.
Les slots riches utilisent des **directives marqueurs** (attribut ou élément) à
importer dans le composant hôte.

```html
<ui-card header="Titre" subheader="Sous-titre">
  Contenu de la carte (slot par défaut)
</ui-card>

<ui-card>
  <img uiCardMedia src="…" alt="…" />
  <span uiCardTitle>Titre <ui-tag label="New" /></span>
  <span uiCardSubtitle>Sous-titre</span>
  Contenu riche…
  <div uiCardFooter>
    <ui-button label="Annuler" level="low" />
    <ui-button label="Valider" />
  </div>
</ui-card>
```

## Contenu full-bleed (`contentFlush`)

Par défaut le contenu porte la gouttière de la carte. `contentFlush` la **retire**
pour laisser le consommateur maîtriser sa mise en page : bande média / couleur
edge-to-edge, tableau pleine largeur… La gouttière reste disponible via la variable
CSS `--ui-card-padding` (à ré-appliquer sur les blocs qui doivent rester en retrait).

```html
<ui-card header="Titre" [contentFlush]="true">
  <div class="flex-y" style="--flex-gap-y: var(--units-md)">
    <div class="banniere"></div> <!-- pleine largeur -->
    <p style="padding-inline: var(--ui-card-padding)">Paragraphe regouttiéré…</p>
  </div>
</ui-card>
```

## Footer

```html
<!-- actions groupées à droite -->
<div uiCardFooter class="flex-x flex-gap-x align-right" style="--flex-gap-x: var(--units-sm)">
  <ui-button label="Annuler" level="low" />
  <ui-button label="Valider" />
</div>

<!-- tags à gauche, action à droite -->
<div uiCardFooter class="flex-x align-justify" style="flex:1;align-items:center">
  <div class="flex-x flex-gap-x"><ui-tag … /><ui-tag … /></div>
  <ui-button label="Action" />
</div>
```
