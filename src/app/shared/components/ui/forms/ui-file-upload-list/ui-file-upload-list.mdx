# ui-file-upload-list

Ligne de fichier — l'**unité répétée** d'une liste de téléversement (miroir du
composant Figma). [ui-file-upload](?path=/docs/components-ui-forms-ui-file-upload--docs)
en empile une par fichier. Présentationnelle : vignette (images) ou icône de
fichier, nom + taille/état, bouton de suppression. Pendant le téléversement, le
marqueur bascule sur `ui-spinner` et une barre de progression apparaît.
