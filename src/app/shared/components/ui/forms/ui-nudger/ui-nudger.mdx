# ui-nudger

Le composant `ui-nudger` est un **incrémenteur numérique** compact : deux boutons
`ui-button` icon-only (`−` / `+`) encadrant la valeur courante. Il n'a pas de champ
de saisie, c'est un pas-à-pas pur, idéal pour de petites quantités (panier, filtres,
paramètres). La valeur est portée via `ControlValueAccessor` : autonome, `[(ngModel)]`,
Reactive Forms ou Signal Forms.

## Bornes (min / max)

`min` et `max` bornent la valeur. Le bouton concerné est **désactivé** dès que la borne
est atteinte, les états Figma `Min` / `Max` sont donc **dérivés** de la valeur, jamais
des props (idiome Angular). Le pas est réglé par `step`.

## Formatage de la valeur

L'input `formatValue` reçoit la valeur numérique et retourne la chaîne affichée
(unité, symbole, etc.).

<Canvas of={UiNudgerStories.Formatted} />

```html
<ui-nudger [(ngModel)]="poids" [min]="0" [formatValue]="(v) => v + ' kg'" ariaLabel="Poids" />
```
