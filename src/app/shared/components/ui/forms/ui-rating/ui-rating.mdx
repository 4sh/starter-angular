# ui-rating

Le composant `ui-rating` permet à l'utilisateur de donner une note sur une échelle (par défaut 5 étoiles). Il est construit sur un `<input type="range">` invisible, garantissant l'accessibilité native (clavier, lecteur d'écran, formulaire).

<Canvas of={UiRatingStories.Default} />

## API

<ArgTypes />

### Tailles
La taille des étoiles et leur espacement sont gérés nativement via l'input `size` (`sm` à `xl`).

## Templates Personnalisés
Vous pouvez remplacer l'icône étoile par n'importe quelle autre icône ou SVG en projetant les templates `#onIcon` et `#offIcon`.
