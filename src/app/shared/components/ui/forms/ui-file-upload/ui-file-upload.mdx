# ui-file-upload

Téléverseur de fichiers **headless** — champ compact (`field`) ou zone de
**glisser-déposer** (`drag`). La sélection s'appuie sur un `<input type="file">`
natif (masqué mais focusable) : sélecteur système, clavier et lecteurs d'écran
fonctionnent nativement, et toute la surface accepte le dépôt. Les fichiers sont
validés côté client (type / taille / nombre) puis téléversés automatiquement
(`auto`), à la demande, ou via un gestionnaire personnalisé (`customUpload`). La
progression est suivie via `XMLHttpRequest` (aucun provider HTTP global requis) ;
les vignettes d'images utilisent des object URLs révoqués à la suppression.

## Custom Upload

L'implémentation du téléversement peut être remplacée en activant `customUpload`
et en définissant un gestionnaire personnalisé (`uploadHandler`). Le gestionnaire
reçoit les fichiers en attente et des callbacks (`setProgress` / `markUploaded` /
`markError`) pour piloter l'état.

<Canvas of={FileUploadStories.CustomUpload} />

## Image Preview

Aperçu en grille avec vignettes + nom / taille. Au survol d'une image, le bouton
de suppression apparaît. Réalisé via le template `content`.

<Canvas of={FileUploadStories.ImagePreview} />

## Template

L'UI est personnalisable via des templates. `file` (variable implicite : le
fichier) remplace le rendu d'une ligne. `content` reçoit les fichiers sélectionnés
et permet de gérer les fichiers téléversés (ex. suppression). `toolbar` affiche du
contenu personnalisé dans la barre d'actions.

```html
<!-- Téléversement réel vers un endpoint -->
<ui-file-upload
  mode="drag"
  [multiple]="true"
  accept="image/*,.pdf"
  [maxFileSize]="5 * 1024 * 1024"
  url="/api/upload"
  ariaLabel="Téléverser des documents"
  (onUpload)="onUploaded($event)"
  (onError)="onError($event)"
/>
```

## Composition

La liste des fichiers est composée de lignes **`ui-file-upload-list`** (une par
fichier) — la même unité que dans Figma. Chaque ligne bascule sur `ui-spinner`
pendant le téléversement. Voir [ui-file-upload-list](?path=/docs/components-ui-forms-ui-file-upload-list--docs).
