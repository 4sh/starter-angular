## Accessibilité

<table className="doc-table">
  <thead>
    <tr>
      <th>Point</th>
      <th>Implémentation</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><strong>Rôle</strong></td>
      <td>Input natif <code>type="checkbox"</code> + <code>role="switch"</code> et <code>aria-checked</code> : annoncé « activé / désactivé ».</td>
    </tr>
    <tr>
      <td><strong>Clavier</strong></td>
      <td>Natif : <kbd>Tab</kbd> pour cibler, <kbd>Espace</kbd> pour basculer. Focus visible via <code>:focus-visible</code> (anneau distinct du hover).</td>
    </tr>
    <tr>
      <td><strong>Nom accessible</strong></td>
      <td>Le clic sur le label (natif, via <code>&lt;label&gt;</code>) bascule le switch. Sans label visible, renseigner <code>ariaLabel</code> (ou <code>ariaLabelledBy</code>).</td>
    </tr>
    <tr>
      <td><strong>Désactivé</strong></td>
      <td><code>disabled</code> natif (pas seulement visuel) ; couleurs via tokens <code>--form-*-disabled</code>.</td>
    </tr>
    <tr>
      <td><strong>Erreur</strong></td>
      <td><code>aria-invalid</code> + bordure erreur ; automatique quand le contrôle est invalide et touché/modifié.</td>
    </tr>
  </tbody>
</table>
