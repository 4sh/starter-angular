# ui-avatar-group

Empile un ensemble de `ui-avatar` avec un léger chevauchement. C’est un simple **conteneur de
mise en page headless** : il n’arrange que ses enfants projetés. Le chevauchement provient de la
constante partagée `$avatar-group-overlap` (`src/styles/src/settings/_ui-config.scss`).

<Canvas of={GroupStories.Default} />

## Débordement « +N »

Le composant ne calcule pas de troncature : exprimez le surplus en projetant un dernier avatar
en mode Label (ex. `+5`).

## Accessibilité

<table className="doc-table">
  <thead>
    <tr>
      <th>Point</th>
      <th>Implémentation</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><strong>Regroupement</strong></td>
      <td>La racine porte <code>role="group"</code>. Chaque <code>ui-avatar</code> conserve son propre nom accessible.</td>
    </tr>
    <tr>
      <td><strong>Avatar « +N »</strong></td>
      <td>Renseigner <code>ariaLabel</code> (ex. « 5 autres membres ») pour décrire le surplus.</td>
    </tr>
  </tbody>
</table>

```html
<ui-avatar-group>
  <ui-avatar image="/a.jpg" alt="Alice" />
  <ui-avatar image="/b.jpg" alt="Bob" />
  <ui-avatar label="+5" ariaLabel="5 autres membres" />
</ui-avatar-group>
```
