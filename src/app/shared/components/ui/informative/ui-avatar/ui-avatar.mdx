# ui-avatar

Représente un utilisateur ou une entité via une **image**, des **initiales** ou une **icône**.
Composant headless, stylé exclusivement par les design tokens (`--informative-defaultlow-*`,
`--global-high-stroke-default`, `--radius-full`…).

Le **mode d’affichage est déduit des inputs** (`image` > `label` > `icon`) — pas de prop `type`
redondante. Un emplacement projeté `[avatarBadge]` superpose un indicateur de statut
(typiquement un `ui-badge`) dans le coin supérieur droit.

## Repli image cassée

Si l’image échoue au chargement, l’avatar bascule automatiquement sur le `label` (ou l’icône) et
émet `imageError`.

## Accessibilité

<table className="doc-table">
  <thead>
    <tr>
      <th>Point</th>
      <th>Implémentation</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><strong>Mode Image</strong></td>
      <td>Le nom accessible vient de <code>alt</code> (ou <code>ariaLabel</code> en repli). Une <code>alt</code> vide rend l’image décorative.</td>
    </tr>
    <tr>
      <td><strong>Mode Label</strong></td>
      <td>La racine porte <code>role="img"</code> + <code>aria-label</code> (<code>ariaLabel</code> sinon les initiales) ; les initiales visibles sont <code>aria-hidden</code> pour éviter la double lecture.</td>
    </tr>
    <tr>
      <td><strong>Mode Icon</strong></td>
      <td>Avec <code>ariaLabel</code> : <code>role="img"</code> + nom. Sans nom : avatar purement décoratif (<code>aria-hidden</code>, un warning est émis en dev).</td>
    </tr>
    <tr>
      <td><strong>Badge</strong></td>
      <td>Le statut porté par la couleur doit être doublé d’un texte (ex. <code>ariaLabel="En ligne"</code> sur le <code>ui-badge</code>).</td>
    </tr>
  </tbody>
</table>

```html
<!-- Avatar image avec pastille de notifications -->
<ui-avatar image="/me.jpg" alt="Jane Doe">
  <ui-badge avatarBadge level="highlight" [value]="3" ariaLabel="3 notifications" />
</ui-avatar>

<!-- Initiales + point de présence -->
<ui-avatar label="JD" ariaLabel="Jane Doe — en ligne">
  <ui-badge avatarBadge level="success" size="small" ariaLabel="En ligne" />
</ui-avatar>
```
