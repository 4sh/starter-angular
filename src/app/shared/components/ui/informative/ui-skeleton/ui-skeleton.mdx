# ui-skeleton

Placeholder affiché **à la place du contenu** pendant son chargement. Rend une
boîte unique colorée par les tokens (`--global-low-surface-default`,
surchargeable via `--ui-skeleton-background`). Trois formes (`text`, `circle`,
`rectangle`) fournissent des dimensions par défaut ; `width` / `height` /
`borderRadius` acceptent une valeur CSS brute pour composer cartes, grilles,
listes et tableaux. La motion respecte `prefers-reduced-motion` et
l'interrupteur global `data-motion="off"`.

## Shapes

Formes et tailles variées via `shape`, `width`, `height`, `borderRadius` et
`size`.

<Canvas of={SkeletonStories.Shapes} />

```html
<ui-skeleton shape="text" />
<ui-skeleton shape="rectangle" />
<ui-skeleton shape="circle" />
<ui-skeleton shape="rectangle" size="small" />
<ui-skeleton shape="rectangle" width="10rem" height="4rem" borderRadius="8px" />
```

## Color

La couleur de fond se personnalise par instance via la custom property
`--ui-skeleton-background`.

## Animation

L'animation de base est **`wave`** : un reflet clair, large et doux, glisse de
gauche à droite et s'estompe aux bords (teinte pilotable via
`--ui-skeleton-shine`, atténuée automatiquement en thème sombre). `pulse`
propose à la place une pulsation d'opacité ; `none` fige la boîte. Toutes
s'arrêtent sous `prefers-reduced-motion` ou `data-motion="off"`.

```html
<ui-skeleton animation="wave" />
<ui-skeleton animation="pulse" />
<ui-skeleton animation="none" />
```

    <tr>

    </tr>
    <tr>
      <td><strong>Clavier</strong></td>
      <td>Aucun élément interactif : le composant n'est jamais dans l'ordre de tabulation.</td>
    </tr>
    <tr>
      <td><strong>Motion réduite</strong></td>
      <td><code>prefers-reduced-motion: reduce</code> et <code>data-motion="off"</code> figent l'animation.</td>
    </tr>
  </tbody>
</table>

```html
<!-- Zone en cours de chargement -->
<div [attr.aria-busy]="loading">
  @if (loading) {
    <ui-skeleton width="100%" />
    <ui-skeleton width="80%" />
  } @else {
    <!-- contenu -->
  }
</div>
```
