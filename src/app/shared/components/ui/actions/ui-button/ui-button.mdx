## Mode lien

Quand un bouton doit **naviguer** (route interne ou URL), renseigner `href` ou
`routerLink` : l'hôte rend alors un **vrai `<a>`** stylé comme un bouton. Le
comportement natif de l'ancre est préservé (Cmd/Ctrl+clic, clic-molette, « ouvrir /
copier le lien »), contrairement à une navigation déclenchée en JS dans un `(click)`.

> Règle : on choisit l'élément par la **sémantique** (naviguer ⇒ `<a>`, agir ⇒
> `<button>`), pas par l'apparence. Pour un lien **inline** dans du texte, préférer
> [`ui-link`](?path=/docs/components-ui-actions-ui-link--docs).

```html
<ui-button label="Voir la page" href="/tarifs" icon="arrow-right" iconPos="right" />
<ui-button label="Mon profil" [routerLink]="['/profil']" />
<ui-button label="Site externe" href="https://angular.dev" target="_blank" />
```
