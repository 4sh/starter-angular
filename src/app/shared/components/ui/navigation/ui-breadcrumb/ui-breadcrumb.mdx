# ui-breadcrumb

Fil d'Ariane **headless** : indique la position courante dans une hiérarchie de navigation.
Chaque élément est une **instance `ui-link`** (composition , pas de styles dupliqués), le
séparateur et le bouton ellipsis consomment les tokens `global` / `actions` (light **et** dark).

## Basic

La position courante dans la hiérarchie : le dernier élément représente la page courante
(<code>aria-current="page"</code>) , il n'a typiquement pas de destination.

```html
<ui-breadcrumb
  [items]="[
    { icon: 'house', ariaLabel: 'Accueil', url: '/' },
    { label: 'Électronique', url: '/electronique' },
    { label: 'Clavier sans fil' },
  ]"
/>
```

## Route

Le fil s'utilise avec le routeur Angular pour naviguer entre les pages : `routerLink`
(chaîne ou tableau de segments) bascule chaque élément sur la directive `RouterLink`
(navigation SPA, comportement natif de l'ancre préservé).

<Canvas of={BreadcrumbStories.Route} />

```html
<ui-breadcrumb
  [items]="[
    { icon: 'house', ariaLabel: 'Accueil', routerLink: '/' },
    { label: 'Catalogue', routerLink: '/catalogue' },
    { label: 'Clavier sans fil' },
  ]"
/>
```

## Custom Separator

Le séparateur se personnalise via l'input `separator` (chaîne) ou, pour un contenu riche,
via le template `#separator` :

<Canvas of={BreadcrumbStories.CustomSeparator} />

```html
<ui-breadcrumb [items]="items">
  <ng-template #separator>
    <ui-icon name="chevron-right" size="sm" />
  </ng-template>
</ui-breadcrumb>
```

## Ellipsis

Une ellipse peut indiquer des éléments masqués : au-delà de `maxItems`, le milieu du fil
se replie derrière un **bouton** ellipsis (premier élément + « … » + éléments de fin).
L'activer révèle les éléments masqués et déplace le focus sur le premier élément révélé ,
les pages intermédiaires restent atteignables au clavier et au lecteur d'écran.

<Canvas of={BreadcrumbStories.Ellipsis} />

```html
<ui-breadcrumb [items]="items" [maxItems]="3" />
```

## Custom Item

Le contenu des éléments se personnalise via le template `#item` (contexte : `$implicit` =
item, `last`) , par exemple pour rendre la page courante en texte simple :

<Canvas of={BreadcrumbStories.CustomItem} />

```html
<ui-breadcrumb [items]="items">
  <ng-template #item let-item let-last="last">
    @if (!last) {
      <ui-link [label]="item.label" [href]="item.url" size="small" />
    } @else {
      <span class="current">{{ item.label }}</span>
    }
  </ng-template>
</ui-breadcrumb>
```

    <tr>

    <tr>
